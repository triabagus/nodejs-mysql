# How to Database MYSQL with Node JS
How to Connect and crud data mysql with node JS.
# Structure Directory 
```
nodejs-mysql/
├── create_db.js
├── create_table.js
├── db_config.js
├── delete.js
├── insert.js
├── insert_multi.js
├── read.js
└── update.js
```
# Installation
1. General | if you not yet install node js and npm
```bash
    lsb_release -a // cek version 
    sudo apt install nodejs
    sudo apt install npm
    npm init // make npm project
```
2. Module
```bash
    npm install mysql
```
# Running
```bash
    node <name_file>
```
